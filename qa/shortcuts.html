Przydatne skróty klawiszowe

Edytory tekstu

Zalecam od razu zobaczyć jak działają poszczególne opcje, a najlepiej uczyć się jednej na dwa dni.

Jeżeli chcesz szybko modyfikować tekst:

  • Ctrl + C
  • Ctrl + V
  • Ctrl + X
  • Backspace
  • Delete
  • Tab - dodaj wcięcie
  • Shift + Tab - zmniejsz wcięcie
  • Ctrl + strzałka lewo/prawo - przeskakujesz pomiędzy poszczególnymi wyrazami
  • Shift + strzałka lewo/prawo - zaznaczasz i odznaczasz kolejny znak
  • Ctrl + Shift + strzałka lewo/prawo - zaznaczasz i odznaczasz kolejne wyrazy
  • Ctrl + Backspace - usuwasz cały wyraz na lewo
  • Ctrl + Delete - usuwasz cały wyraz na prawo
  • Home - przesuwasz się na początek linii
  • End - przesuwasz się na koniec linii
  • Ctrl + Home - przesuwasz się na początek dokumentu
  • Ctrl + End - przesuwasz się na koniec dokumentu
  • Shift + Home - zaznaczasz od danego miejsca do początku linii
  • Shift + End - zaznaczasz od danego miejsca do końca linii
  • Ctrl + Shift + Home - zaznaczasz od danego miejsca do początku dokumentu
  • Ctrl + Shift + End - zaznaczasz od danego miejsca do końca dokumentu
  • Shift + Insert - wklej, działa w konsoli, jak i w innych programach

Konsola Linuxa:

Nie wszystkie polecenia z konsoli będą łatwe/sensowne do nauczenia/użycia :)
  • Shift + Insert - wklej
  • Ctrl + L - wyczyść konsolę
  • Ctrl + A - przejdź na początek linii (pomoże jak zapomnisz o sudo)
  • Ctrl + E - przejdź na koniec linii
  • Ctrl + U - usuń cały tekst
  • Ctrl + K - usuń wszystko na prawo od kursora
  • Ctrl + W - usuń wyraz na lewo
  • Ctrl + Y - wklej tekst usunięty przez Ctrl + W/K/U
  • Strzałka góra/dół - wyświetla wcześniejsze/późniejsze zatwierdzone polecenie
  • Ctrl + R - wyszukiwanie wśród historii, ponowne wciśnięcie Ctrl + R pokazuje następną pozycję w historii
  • !! - powtarza poprzednie polecenie, przydatne z sudo (sudo !!)

Chrome

  • Ctrl + F
  • Ctrl + H
  • Ctrl + J
  • Ctrl + S
  • Ctrl + 1-8
  • Ctrl + 9
  • Ctrl + W - zamknij kartę
  • Ctrl + N - otwórz nowe okno
  • Ctrl + Shift + N - tryb porno
  • Ctrl + T - nowa karta
  • Ctrl + Shift + T - otwórz poprzednio zamkniętą kartę
  • Ctrl + L - przejdź do paska URL
  • Ctrl + Shift + B - ukryj pasek zakładek
  • Ctrl + U - wyświetl kod źródłowy strony
  • Page Up/Page Down - strona wyżej/niżej
  • Home - początek strony
  • End - koniec strony

Windows 10

  • Windows + M
  • Alt + F4
  • Alt + Ctrl + Del
  • Windows + L - zablokuj konto
  • Shift + Del - bardzo potężne polecenie, ostrożnie
  • Ctrl + Shift + N - tworzy nowy folder
  • F2 - zmiana nazwy zaznaczonego pliku
  • Ctrl + Shift + Esc - uruchamia Menedżera zadań
  • Alt + Tab - zmiana na poprzednie/kolejne okno
  • Alt + Shift + Tab - zmiana na wcześniejsze okno
  • Sift + Tab - wyświetl wszystkie okna, z tego miejsca można utworzyć wirtualny pulpit
  • Ctrl + Windows + strzałka lewo/prawo - przejdź do kolejnych pulpitów wirtualnych (jeśli utworzone)
  • Windows + E - eksplorator Windows
  • Alt + Enter - wyświtla właściwości zaznaczonego pliku
  • Windows + Pause - właściwości systemu
  • Shift + F10 - wyświetla menu kontekstowe
  • Shift + Prawy przycisk myszy - pozwala otworzyć menu z dodatkową opcją PowerShell w danym katalogu

Klikając prawym przyciskiem myszy na ikonę Windowsa, otrzymujemy multum przydatnych narzędzi.
